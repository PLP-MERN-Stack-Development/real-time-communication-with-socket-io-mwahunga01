# Real-Time Chat Application with Socket.io

A fully functional real-time chat app built with **React + Vite + Tailwind CSS** (frontend) and **Node.js + Express + Socket.io** (backend).

## Live Demo
https://plp-socketio-chat-2025.netlify.app

Backend Server: https://plp-chat-server.up.railway.app

## Features Implemented
- Real-time messaging (instant delivery)
- User join/leave notifications
- Online users list
- Typing indicators ("X is typing...")
- Private messaging (DMs)
- Message history persistence
- Beautiful, responsive UI
- Auto-reconnect on disconnect

## Tech Stack
- Frontend: React 18 + Vite + Tailwind CSS
- Backend: Node.js + Express + Socket.io
- Real-time: Socket.io with WebSocket + fallback
- Deployment: Netlify (client), Railway (server)
